## 使用方法

### 1. 手动测试

在前端界面逐个输入上述短语，检查生成质量。

### 2. 批量测试（脚本）

使用测试脚本：

```bash
./test_api.sh
```

### 3. 自定义测试

创建 `custom_test.json`:

```json
{
  "phrases": [
    "你的测试短语1",
    "你的测试短语2"
  ]
}
```

## 质量评估标准

### 翻译准确性（1-5分）

- 5分：完全准确，符合母语习惯
- 4分：准确，略有生硬
- 3分：基本准确，有明显翻译腔
- 2分：部分准确，有错误
- 1分：严重错误或误译

### 例句质量（1-5分）

- 5分：自然口语，场景真实，可直接使用
- 4分：较自然，场景合理
- 3分：可理解，但略显造作
- 2分：生硬，不符合日常表达
- 1分：语法错误或完全不自然

### 日语注音正确性（1-5分）

- 5分：所有汉字正确注音，外来语标注完整
- 4分：大部分正确，个别遗漏
- 3分：部分正确，有明显遗漏
- 2分：注音错误较多
- 1分：大量错误或缺失
