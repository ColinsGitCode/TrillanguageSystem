## 5. 发现的问题（需修复）

### P1：Knowledge Hub 的 synonym detail 接口异常
- 现象：`/api/knowledge/synonyms/list` 有数据，但请求 `/api/knowledge/synonyms/:pairKey` 返回 `404 {"error":"synonym boundary not found"}`。
- 影响：关系详情面板无法打开，Hub 页面功能不完整。
- 建议：统一 list 与 detail 的 key 生成/编码策略，清洗 `pairKey='...'` 这类无效数据。

### P2：自动化 UI 工具在删除按钮处交互不稳定
- 现象：MCP Chrome DevTools 对弹窗垃圾桶按钮多次点击超时，后续出现 `Target closed`。
- 影响：自动化回归脚本稳定性下降；人工删除不一定受影响。
- 建议：
  - 检查删除按钮层级与可交互区域（z-index、pointer-events、遮罩）；
  - 给删除确认流程增加稳定的可测试选择器与显式状态提示；
  - 增加 Playwright 兜底回归脚本。
